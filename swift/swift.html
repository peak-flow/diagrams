Swift App Lifecycle

Not Running

App isn't launched or was terminated by the system

Inactive

Running in foreground but not receiving events (e.g., during transitions)

Active

Running in foreground and receiving events

Background

Running code in the background, not visible to user

Suspended

In memory but not executing code, can be terminated without notice

Key Lifecycle Methods

Framework Method Description
UIKit application(_:didFinishLaunchingWithOptions:) Called when the app launches, for one-time setup
UIKit sceneWillEnterForeground(_:) Scene is about to begin running in the foreground
UIKit sceneDidBecomeActive(_:) Scene is now active and receiving events
UIKit sceneWillResignActive(_:) Scene is about to stop receiving events
UIKit sceneDidEnterBackground(_:) Scene has entered the background
SwiftUI init() Initialize app and state
SwiftUI onAppear View appeared on screen
SwiftUI onDisappear
